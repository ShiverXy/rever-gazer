//////////////////////////////////////////////////////////////////////////////
SYSTEM REQUIREMENTS
//////////////////////////////////////////////////////////////////////////////

200 Mhx Pentium or higher
Win' 98 or higher
30 MB hard disk space, 32 MB RAM (256 MB for XP and above)
Windows-compatible sound card and speakers

//////////////////////////////////////////////////////////////////////////////
CONTROLS
//////////////////////////////////////////////////////////////////////////////

  - - - - - - - - - - - 
|    _ _ _   _ _ _     |
| A |     | |     |  1 |
| B |     | |     |  2 |
| C |_ _ _| |_ _ _|  3 |
|      ?       !       |
  - - - - - - - - - - - 
  
BUTTON A
 - Preview: Shows off some sprites.
 
 BUTTON B
 - Music: Shows off some songs.
 
 BUTTON C
 - Cancel: Quits the application.

BUTTON 1
 - Cycle: Scrolls through a selection, select with 2.

BUTTON 2
 - Select: Plays a cry or song.

BUTTON 3
 - Cancel: Backs out of a prompt.

Button ?
- Does nothing.

Button !
- Does nothing.
